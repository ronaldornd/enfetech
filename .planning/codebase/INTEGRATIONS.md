# Codebase Integrations

## External Services
- **Google Gemini API**: Integrated via `@google/genai`. API Key is managed through environment variables (`GEMINI_API_KEY`) and injected via Vite's `define` config.

## Build/Tooling Integrations
- **Tailwind CSS v4**: Integrated as a Vite plugin (`@tailwindcss/vite`).
- **Framer Motion**: Integrated for advanced UI animations (imported as `motion`).
- **Lucide React**: Used for the comprehensive icon system.
