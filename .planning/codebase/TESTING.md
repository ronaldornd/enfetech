# Codebase Testing Reference

## Current State
- **Automated Tests**: No unit or integration tests (Vitest/Jest) are currently implemented.
- **Verification Method**: Manual validation through the Vite dev server.
- **Type Safety**: TypeScript (`tsc`) is used for static analysis and build-time checks.

## Planned Improvements
- Implementation of Vitest for logic testing (Drip rate calculations, XP logic).
- E2E testing for user flows (Modules -> Lessons -> Flashcards).
